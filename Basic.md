**java中有三种移位运算符**

"<<"  左移运算符，num << 1,相当于num乘以2

">>" 右移运算符，num >> 1,相当于num除以2

">>>"  无符号右移，忽略符号位，空位都以0补齐

"<< =" 左移位赋值运算符	C << = 2等价于C = C << 2

">> ="	右移位赋值运算符	C >> = 2等价于C = C >> 2
